# Book Tracker

**Автор:** Линец София Николаевна

## Описание программы
Book Tracker — это GUI-приложение на Python (Tkinter) для управления списком прочитанных книг. Программа позволяет добавлять книги, просматривать их в таблице, фильтровать по жанру и количеству страниц, а также сохранять и загружать данные через JSON.

## Как использовать
1. Установите Python 3.
2. Скачайте или клонируйте проект.
3. Перейдите в папку проекта.
4. Запустите файл:
   ```bash
   python book_tracker.py
   ```
5. Заполните поля: название книги, автор, жанр, количество страниц.
6. Нажмите **Добавить книгу**.
7. При необходимости примените фильтр по жанру или по количеству страниц.
8. Сохраните или загрузите данные через JSON-кнопки.

## Примеры использования
- Добавить книгу: `Чистый код`, `Роберт Мартин`, `Научная`, `464`.
- Ввести пустое название — программа должна показать ошибку.
- Ввести страницы `abc` — программа должна показать ошибку.
- Отфильтровать книги жанра `Детектив`.
- Отфильтровать книги с количеством страниц больше `200`.
- Сохранить список в JSON и загрузить его обратно.
